Министерството за внатрешни работи информира дека утре, на 01.10.2023 година (недела) со почеток во 08.00 часот, поради одржување на Скопски маратон 2023, ќе биде спроведен посебен сообраќаен режим на подрачјето на град Скопје.

Трасата на движење на трката од 5 км ќе започне од Собрание на Република Северна Македонија, право по ул.„11 Октомври“, лево по бул. „Крушевска Република“, десно по бул.„Трета македонска бригада“, полукружно на крстосницата помеѓу бул.„Трета македонска бригада“ и ул.„Коста Новаковиќ“ и назад по истата траса до целта – Собрание на Република Северна Македонија на ул.„11 Октомври“.

Стартот на трасите на движење на полумаратонот на 21 км и штафетната трка на 42 км, при што по наведената траса за штафетната трка на 42 км натпреварувачите се движат два пати, е: Собрание на Република Северна Македонија на ул.„11 Октомври“, право кон плоштад „Македонија“ низ Порта Македонија, лево по ул.„Никола Вапцаров“, десно по ул.„Димитрие Чуповски“, лево по бул.„Св.Климент Охридски“, десно по ул.„Св.Кирил и Методиј“, право по ул.„Митрополит Теодосиј Гологанов“, право по ул.„Војвода Васил Чакаларов“, сѐ до крстосницата со ул.„Исаија Маџовски“, каде се врти полукружно назад по истата траса сѐ до Собрание на Република Северна Македонија на ул.„11 Октомври“ и се продолжува право по ул.„11 Октомври“, лево по бул.„Крушевска Република“, десно по бул.„Трета македонска бригада“, лево по бул.„Србија“ (во спротивен правец), лево по бул.„Јане Сандански“ (во спротивен правец), сѐ до крстосницата со ул.„Васко Карангелевски“, каде се врти полукружно по истите улици до целта – плоштад „Македонија“.

Од тие причини Единицата за безбедност на патниот сообраќај ќе преземе мерки за посебен режим на сообраќајот, при што сообраќајот ќе се пренасочува кон околните сообраќајници.

МВР апелира до граѓаните да ги почитуваат наредбите што ги издаваат полициските службеници по сообраќајниците каде што ќе се одвива трката.
